# Fotografen Portfolio - JW Media

Eine moderne, responsive Website für ein Fotografen-Portfolio mit Next.js, React und Tailwind CSS.

## Features

✨ **Modern & Minimalistisch**
- Dark Theme Design
- Responsive Layout (Mobile, Tablet, Desktop)
- Smooth Animations mit Framer Motion

📸 **Funktionen**
- Galerie mit Fotokategorien
- Ausgewählte Projekte
- Kontaktformular
- Direkte Kontaktoptionen
- Optimiert für GitHub Pages

## Installation

```bash
# Dependencies installieren
npm install

# Entwicklungsserver starten
npm run dev

# Für GitHub Pages bauen
npm run export
```

## Technologie Stack

- **Next.js 14** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animationen
- **GitHub Pages** - Hosting

## GitHub Pages Setup

1. Gehe zu deinen Repository Settings
2. Scrolle zu "GitHub Pages"
3. Wähle "Deploy from a branch"
4. Wähle `main` branch und `/root` folder
5. Speichern

Deine Website wird dann unter `https://distortedheights.github.io/jwmedia` verfügbar sein.

## Anpassungen

- Bilder in `public/images` hinzufügen
- Texte in den Komponenten bearbeiten
- Farben in `tailwind.config.js` anpassen
- Kontaktformular mit deinem Backend verbinden
